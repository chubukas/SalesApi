# SalesAppAPI

A Sales Api

APIs:

Base Endpoint : 
http://localhost:3000/

GET Endpoints : 

1. /daily : To check daily sales by hour
2. /weekly : To check sales by days of the week
3. /monthly : To check sales by days of the monthly

POST Endpoint :

1. /addSales : To add data to sales table
   
    Payload: 
            {

               username (string)
               amount (number)
             }
